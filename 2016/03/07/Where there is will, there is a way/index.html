Where there is will, there is a way

發表於 2016-03-07

写在妇女节的前一天。
祝今天的女生,女生节快乐,明天的妇女,妇女节快乐.
仔细想来,从开始学习前端到目前也已经有9个月了.其中酸楚也只有自己才知道.毕竟工作以后不能像读书时候那样任性了,没有那么多的时间,并且自己还要赚钱养活自己.
从一开始就对这个行业充满期待,憧憬,觉得之后世界都将是程序员的,到自己一步步的慢慢接触各种知识,自己一步步的动手做网站,写脚本,扶墙上网,装系统,调试bug,在这个过程中,我找到了好久都没有找到的那种愉悦,也更加确信了自己之后将坚持的这条路.
迟来的,总是最好的
以前一直为自己并不是计算机及相关专业<\q>毕业的而感到深深的自卑,而自己也因为这个门槛,不敢往里迈入一步.总觉得自己不能浪费了自己大学四年所学的专业.但是在不知道经历了多少的曲折之后,我才认识到几个非常严重的问题.

强弩之末势不能穿缟

说的现实点,于我专业相关的这些化工专业就是强弩之末,自己在这上边拼命的挣扎一辈子,也许也不如别人努力一十年.反倒让自己早早‘吸太多毒气’而身体衰微.当我有了这个念想之后,就开始着手这改变自己的主攻专业了.之前所学的专业知识,不过都是沉没成本.理性人,不考虑沉默成本.

日本经济发展史

我有一段时间长期出差在外,而正好那段日子并没有什么太多的工作要做,于是自己变拿起了自己很感兴趣的经济学读物又看了起来.有一本可能不太专业的书,叫《日本经济的盛衰》.看完之后给我的启发挺大的.
1.所谓的中国经济飞速发展的奇迹,日本也有,韩国也有,新加坡也有,台湾也有….很多时候,我们都是照着老路子走罢了
2.这些发达国家发迹过程中几乎都有一个明显的特点,就是产业转型,一开始时粗放型,重工业,化工业,后来便是高科技,文化产业,服务产业
3.在转型的过程中,落后的产业就将被新兴的产业替代
(虽说这些推断还有认识可能在专业选手看来不值一提，甚至错误百出,但当时就是对这几点深信不疑的)
于是我猛的就想到了我们国家现在的情况.制造业尾大不掉,产能过剩,而互联网如新兴的朝阳,灿烂的让人不能直视…….在意思到这些时候,我想逆流而上会粉身碎骨,而明智的决定是跟随时代的潮流.

种一棵树最好的时候一个是十年前,另一个是现在

以绝大多人努力程度之低,根本轮不上拼天赋

虽然这两句话看起来很像毒鸡汤,但是确确实实在我迷茫的时候,他们一直激励着我.
我知道前方的路还很长,不过我,准备好了.千山万水,只等我来跨越.
