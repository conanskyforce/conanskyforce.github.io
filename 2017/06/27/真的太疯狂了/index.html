真的太疯狂了

發表於 2017-06-27

最近居然没有好好写博客了！真是不行不行的！一想到自己隔了一段时间没写点东西，就感到十分的不安。觉得自己好像没有存在感了似地。

这个月可以说是高强度的在前端,哦不,暂且可以不要脸的说一下是在全栈的路上高歌猛进了!
虽说大前端栈很多工具用的还不是特别熟练,但觉得自己逐渐能摸到门道了。看到问题,也知道该怎么样去分析解决了,
而不是一味的害怕,无知.最重要的是培养了我自己对这个行业坚定的信心，以及对其广泛的兴趣。

If you don’t know, learn it.我们每个人都是终身学习者.

唯一觉得稍有遗憾的就是自己没有很早的接触编程.

另外，最近这两天疯狂的弥补区块链，比特币，cryptocurrency的相关知识。因为自己算是有一点经济学的基础，经常关注财经的类的节目,书籍,也学过一段时间。虽然自己现在还算不上经济学高手。但是即使是我，也知道区块链，去中心化，即将对金融行业、以及所有涉及交易、公证、信用的行业带来多大的影响。

这里有个ted的视频，这个老头子讲的还挺宽泛。即使是实现其中之一，也是非常了不起的。


有一句话，不知道是谁说的。

对于新兴事物，绝大多数人第一是看不见，第二是看不懂，第三是看不起，第四是来不及。

这句话对于比特币(区块链)来说，真的是再适合不过了。
看好它的人，觉得它能颠覆世界，带来一场伟大的革命。而看不懂的人觉得它就是一场阴谋，一场庞氏骗局。
虽然不敢说自己很懂，但我知道，自己还是很乐于拥抱新鲜事物的，而且也希望自己一直如此。
之后将建个关于比特币/区块链的技术网站。暂定为

bytecoin.tech

希望我们能在这个动荡的时代过的更好。
