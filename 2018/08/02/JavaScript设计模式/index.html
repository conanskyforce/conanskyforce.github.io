简单模拟Object.create

1.工厂模式

无法识别类型,无法复用函数,属性

2.构造函数

构造函数内的函数，不能复用，都是不一样的函数

3.构造函数 + 原型模式

算是比较符合js设计原则的OOP模式


(构造)函数 都有一个prototype属性，指向原型对象,
原型对象都有一个constructor属性指向构造函数,
实例都有一个指向原型对象的指针proto

原型链的继承
